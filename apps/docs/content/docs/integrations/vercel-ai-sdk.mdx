description: Use the Vercel AI SDK as the LLM backend and tool layer for FitalyAgents.

The [Vercel AI SDK](https://sdk.vercel.ai) (`ai` package) provides a unified interface over multiple LLM providers (OpenAI, Anthropic, Google, Mistral, and more) with first-class streaming support. You can use it as the **LLM backend for the Dispatcher** or to power individual agents.

## Install

```bash
pnpm add ai
# plus one provider package:
pnpm add @ai-sdk/openai        # OpenAI / Azure
pnpm add @ai-sdk/anthropic     # Claude
pnpm add @ai-sdk/google        # Gemini
```

## LLM provider for the Dispatcher

Implement `LLMProvider` using `generateText`. The single method is `complete(system, user): Promise<string>`:

```typescript
import { type LLMProvider } from 'fitalyagents/dispatcher'
import { generateText } from 'ai'
import { openai } from '@ai-sdk/openai'

export class VercelAIProvider implements LLMProvider {
  constructor(private model = openai('gpt-4o-mini')) {}

  async complete(system: string, user: string): Promise<string> {
    const { text } = await generateText({
      model: this.model,
      system,
      prompt: user,
    })
    return text
  }
}
```

Swap the model at construction time to change providers without touching the rest of your code:

```typescript
import { anthropic } from '@ai-sdk/anthropic'
import { google } from '@ai-sdk/google'

// Claude
new VercelAIProvider(anthropic('claude-3-5-haiku-20241022'))

// Gemini
new VercelAIProvider(google('gemini-2.0-flash'))
```

Wire it into `DispatcherBootstrapper` and start `NodeDispatcher`:

```typescript
import {
  NodeDispatcher,
  InMemoryEmbeddingClassifier,
  InMemoryIntentLibrary,
  DispatcherBootstrapper,
} from 'fitalyagents/dispatcher'

const intentLibrary = new InMemoryIntentLibrary()
const classifier = new InMemoryEmbeddingClassifier({ intentLibrary })

const bootstrapper = new DispatcherBootstrapper({
  intentLibrary,
  llm: new VercelAIProvider(),
})

await bootstrapper.bootstrapFromManifests([
  interactionAgent.manifest,
  workAgent.manifest,
  orderAgent.manifest,
])

await classifier.init()

const dispatcher = new NodeDispatcher({ bus, classifier, intentLibrary })
await dispatcher.start()
```

## Use AI SDK tools inside an agent

Register tools from the AI SDK in `ToolRegistry` using `registerFunctionHandler`:

```typescript
import { ToolRegistry, ExecutorPool, registerFunctionHandler } from '@fitalyagents/asynctools'
import { tool } from 'ai'
import { z } from 'zod'

const registry = new ToolRegistry()

// Define the tool schema with AI SDK
const getWeatherSchema = z.object({ city: z.string() })

registry.register({
  tool_id: 'get_weather',
  description: 'Get current weather for a city',
  executor: { type: 'ts_fn' },
  timeout_ms: 5000,
})

registerFunctionHandler('get_weather', async (input) => {
  const { city } = getWeatherSchema.parse(input)
  const res = await fetch(`https://wttr.in/${city}?format=j1`)
  const data = await res.json()
  return { temperature: data.current_condition[0].temp_C }
})

const pool = new ExecutorPool(registry)
```

Use with `pool.execute()` in any agent's `process()`:

```typescript
import { NexusAgent } from 'fitalyagents'

class WeatherAgent extends NexusAgent {
  manifest = { agent_id: 'weather-agent', capabilities: ['WEATHER'], /* ... */ }

  async process(task) {
    const result = await pool.execute('get_weather', task.task_id, { city: task.slots?.city })

    return {
      status: 'completed',
      result: result.result,
      event: 'TASK_RESULT',
      task_id: task.task_id,
      session_id: task.session_id,
      agent_id: this.manifest.agent_id,
      completed_at: Date.now(),
    }
  }
}
```

## Streaming responses through the bus

For long-running responses, stream partial text events to the bus so subscribers can display incremental output:

```typescript
import { streamText } from 'ai'
import { openai } from '@ai-sdk/openai'
import { NexusAgent } from 'fitalyagents'

class StreamingAgent extends NexusAgent {
  manifest = { agent_id: 'streaming-agent', capabilities: ['CHAT'], /* ... */ }

  async process(task) {
    const { textStream } = streamText({
      model: openai('gpt-4o-mini'),
      prompt: task.slots?.message as string,
    })

    let fullText = ''

    for await (const chunk of textStream) {
      fullText += chunk
      await this.bus.publish('bus:AGENT_STREAM_CHUNK', {
        session_id: task.session_id,
        agent_id: this.manifest.agent_id,
        chunk,
      })
    }

    return {
      status: 'completed',
      result: { text: fullText },
      event: 'TASK_RESULT',
      task_id: task.task_id,
      session_id: task.session_id,
      agent_id: this.manifest.agent_id,
      completed_at: Date.now(),
    }
  }
}
```

Subscribe on the client or interaction layer:

```typescript
bus.subscribe('bus:AGENT_STREAM_CHUNK', ({ session_id, chunk }) => {
  // Push to WebSocket, SSE, or TTS pipeline
  ttsStream.write(chunk)
})
```

## Structured output with `generateObject`

Use `generateObject` to get typed, schema-validated results from an LLM inside an agent:

```typescript
import { generateObject } from 'ai'
import { openai } from '@ai-sdk/openai'
import { z } from 'zod'

const OrderSchema = z.object({
  items: z.array(z.object({ sku: z.string(), qty: z.number() })),
  delivery_date: z.string().optional(),
})

class ParseAgent extends NexusAgent {
  manifest = { agent_id: 'parse-agent', capabilities: ['ORDER_PARSE'], /* ... */ }

  async process(task) {
    const { object } = await generateObject({
      model: openai('gpt-4o'),
      schema: OrderSchema,
      prompt: `Extract order details from: "${task.slots?.utterance}"`,
    })

    return {
      status: 'completed',
      result: object,
      event: 'TASK_RESULT',
      task_id: task.task_id,
      session_id: task.session_id,
      agent_id: this.manifest.agent_id,
      completed_at: Date.now(),
    }
  }
}
```

## Provider-agnostic agent

Because `LanguageModelV1` is the common interface, you can inject the model at bootstrap time:

```typescript
import { generateText, type LanguageModelV1 } from 'ai'
import { anthropic } from '@ai-sdk/anthropic'
import { NexusAgent, type IEventBus } from 'fitalyagents'

class FlexibleAgent extends NexusAgent {
  constructor(private model: LanguageModelV1, options: { bus: IEventBus }) {
    super(options)
  }

  async process(task) {
    const { text } = await generateText({ model: this.model, prompt: task.slots?.query as string })
    return {
      status: 'completed',
      result: { text },
      event: 'TASK_RESULT',
      task_id: task.task_id,
      session_id: task.session_id,
      agent_id: this.manifest.agent_id,
      completed_at: Date.now(),
    }
  }
}

// At startup — swap without touching agent code:
const agent = new FlexibleAgent(anthropic('claude-3-5-sonnet-20241022'), { bus })
```
